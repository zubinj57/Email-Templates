Since Hilton PEP for LAPLA is functioning very similar to HotelKey, please follow the steps below for the Onboarding Reports.

A. Onboarding Reports

Note: These onboarding reports are to be sent just once during the initial setup phase. We do not need these reports on a recurring basis.
Export steps (apply to each batch):
  1. Run the report
  2. Make sure Show Grand Total is UN-CHECKED
  3. Click the Export icon
  4. Select export type as CSV
  5. Ensure Group By Views are UN-CHECKED and All Columns are SELECTED
  6. Click Export
  7. Rename the file to the date range for identification (examples shown below)
  8. Repeat for the next batch
